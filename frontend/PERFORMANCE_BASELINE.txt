Performance baseline — how to measure (Faz 0)
==============================================

1) Browser (Chrome/Edge)
   - Open the app with profiling: add ?perf=1 to the URL, or run:
     localStorage.setItem("yiroPerf", "1"); location.reload();
   - DevTools → Performance: record while: cold load, login, search typing, Analytics tab, artist detail, queue scroll.
   - DevTools → Network: note slow requests (waterfall, TTFB).
   - In Performance, check User Timings for: yiro-search-fetch, yiro-render-search, yiro-data-refresh (full tab refresh), yiro-refresh-light, yiro-refresh-heavy.

2) Backend (after request timing middleware is deployed)
   - Check server logs for lines like: slow_request path=... duration_ms=...
   - Note which API paths exceed 300–500 ms under normal load.
